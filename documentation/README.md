# GitHub Cruise - Documentation

This folder contains technical documentation for the project.

## Documentation Files

### FEATURES.md
Complete feature documentation covering:
- Core features (Splash, Search, Profiles, Repositories)
- UI features (Themes, Localization, Design)
- Technical features (Architecture, DI, Networking)
- Accessibility and developer features

### ARCHITECTURE.md
Technical architecture guide including:
- Architecture overview (Clean Architecture + MVVM)
- Layer architecture details
- Design patterns used
- Dependency injection setup
- State management and data flow
- Testing strategy

### API.md
GitHub API integration details:
- Search Users API
- Get User Profile API
- Get User Repositories API
- Request/response examples

### DESIGN_SYSTEM.md
Material Design 3 implementation:
- Design principles
- Spacing, elevation, shape systems
- Color system (brand and semantic colors)
- Typography system
- Component dimensions
- Usage examples and best practices

### CODE_COVERAGE.md
Code coverage documentation:
- Coverage tool setup (JaCoCo)
- Running coverage reports
- Current coverage statistics
- Coverage goals and roadmap
- Testing best practices

### TESTING_TYPES.md
Complete testing guide:
- All Android test types explained
- Unit, Integration, UI, E2E, Screenshot tests
- Current implementation status
- Official Android documentation links
- Recommended testing strategy
